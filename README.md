# Распознавание открывания пива

- При открытии любой алюминиевой банки запускается приложение, которое нужно указать 

- Реализуется на librosa и scikit-learn

- Модель обучалась на дереве решений, ей были предоставлены 17 целевых файлов и около 40 «шумовых»


# Возможности

- Распознавание открывания пива
- Возможность дообучения модели  


# Установка

 1. Клонируйте репозиторий

```bash
git clone https://https://github.com/fan1omas/beer-detection
cd dino-play
```

 2. Загрузите необходимые библиотеки и заполните condig-файл
```bash
pip install -r requirements.txt
```
 3. Попробуйте: 
 
- Откройте холодненькое
- Наслаждайтесь пивом и играйте в любимую игру
